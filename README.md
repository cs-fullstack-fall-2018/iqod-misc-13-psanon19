## encodeWord

Define a function called ​`encodeWord`​ that receives a string and returns the codified version of that string replacing the following characters:
- `3` instead of `E`
- `1` instead of `I`
- `@` instead of `A`
- `$` instead of `S`
- `0` instead of `O`

Print the result (to console/browser/etc.)
